## Required Narrative Boundary

The video must preserve this line:

```text
报告负责形成假设，事件层负责验证假设。
```

The disease survey report is disease-level research output. It is not a single-ticker event feed. K-line/backtest should be framed as ticker-level trusted event validation. Any simulated/mock curve must be labeled as `模拟演示路径，不代表真实研究收益`, not real research performance.
